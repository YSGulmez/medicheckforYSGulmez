# Medicheck

Medicheck is a mobile application developed with Flutter that allows you to manage your medication easily. 

## Features

- **Medication Management**: Add, edit, and delete your medicines.
- **Tracking**: Record values such as tablet dates and counts.
- **Local Storage**: All data is stored locally using SQLite for privacy and offline access.
- **Offline Capable**: No internet connection required.

## Getting Started

### Prerequisites

- Flutter SDK (3.7.2 or higher)
- Dart SDK

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/YSGulmez/medicheckforYSGulmez.git
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the app**
   ```bash
   flutter run
   ```
