## Reporting a Vulnerability

We take the security of our software very seriously. If you find a security vulnerability, please send an e-mail to **yusufgulmez4535@gmail.com**.

Please do not report security vulnerabilities through public GitHub issues.

## Security Measures

- **Data Encryption**: All local data is stored in a secure SQLite database.
- **No Remote Storage**: This application works completely offline and does not send data to any remote server.
- **Code Audits**: We perform regular code reviews to ensure code quality and security.
